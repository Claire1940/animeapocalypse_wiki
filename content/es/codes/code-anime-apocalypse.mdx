## Por qué fallan los códigos (y cómo solucionarlo rápido)

Incluso los códigos válidos pueden fallar por pequeños problemas técnicos. No entres en pánico: soluciona en este orden:

| Mensaje de error / Problema | Causa más probable | Solución |
|---|---|---|
| **Código inválido** | Error tipográfico, mayúsculas incorrectas, mala fuente | Vuelve a copiar desde una fuente confiable y reintenta |
| **Código vencido** | Terminó la recompensa por tiempo limitado | Elimínalo de la lista y sigue |
| **Ya canjeado** | Ya fue reclamado en tu cuenta | No se necesita ninguna acción |
| **Sin respuesta tras canjear** | Retraso de interfaz/servidor | Espera 10–20 segundos, luego reintenta una vez |
| **El menú de códigos no abre** | Bug de interfaz o lag | Vuelve a entrar al servidor o reinicia el juego |
| **Recompensa no visible** | Tipo de objeto reclamado no claro | Revisa todas las pestañas (moneda, mejoras, inventario) |

### Comprobaciones avanzadas para problemas persistentes

Si aún no puedes reclamar una entrada que funciona:

- Cambia a una región de servidor diferente.
- Desactiva descargas/streams pesados en segundo plano.
- Vuelve a iniciar sesión en tu cuenta de Roblox.
- Prueba solo una entrada nueva de code anime apocalypse después de reconectarte.
- Confirma que el código no fue publicado para un modo de evento limitado en el que no estás.

Estos pasos resuelven la mayoría de problemas del lado del jugador sin necesidad de tickets de soporte.
