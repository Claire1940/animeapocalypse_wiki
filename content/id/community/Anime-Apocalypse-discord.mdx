## Mengatur Makro AFK Anda untuk Kesuksesan

Salah satu sumber daya yang paling dicari di **discord Anime Apocalypse** adalah konfigurasi makro TinyTask. Alat ini memungkinkan Anda merekam serangkaian input dan mengulanginya tanpa batas. Ini sangat berguna untuk "Infinite Mode," di mana tujuannya adalah bertahan dari sebanyak mungkin gelombang untuk mendapatkan unit level maksimal dan aksesori langka.

### Persyaratan Konfigurasi Makro
Untuk memastikan makro Anda berjalan tanpa kesalahan, Anda harus mencocokkan pengaturan sistem Anda dengan persyaratan skrip. Sebagian besar makro yang dibagikan pada tahun 2026 dirancang untuk output standar 1080p.

1.  **Resolusi:** Atur resolusi layar Anda ke **1920 x 1080**. Sebagian besar skrip menggunakan klik berbasis koordinat, sehingga penyimpangan apa pun akan menyebabkan makro mengklik elemen UI yang salah.
2.  **Kecepatan Game:** Pastikan kecepatan dalam game Anda diatur ke **3x atau 4x**. Ini memastikan waktu penggunaan kemampuan makro sesuai dengan kecepatan gelombang musuh.
3.  **Pergerakan:** Makro yang baik harus menyertakan perintah "jalan ke depan". Ini membersihkan area spawn awal, memungkinkan musuh untuk menyerang dan pertandingan berlangsung dengan benar.

> **Peringatan:** Menggunakan makro umumnya diterima untuk farming AFK di mode privat atau infinite, tetapi selalu periksa aturan terbaru di **discord Anime Apocalypse** untuk memastikan Anda tidak melanggar ketentuan layanan saat ini.

## Gadget dan Peningkatan Kemampuan

Untuk menembus gelombang 300 dan mengamankan tempat di papan peringkat, unit Anda membutuhkan gadget yang tepat. Gadget bertindak sebagai pengubah pasif atau aktif yang dapat menjembatani kesenjangan antara sesi yang gagal dan farm semalam yang sukses.

Gadget **Black Hole** saat ini merupakan standar emas untuk mode infinite. Gadget ini menarik musuh ke satu titik pusat, memungkinkan unit AoE Anda seperti Sanji atau Overlord untuk memaksimalkan damage per second (DPS) mereka. Saat mengonfigurasi peningkatan unit Anda di dalam makro, selalu prioritaskan **Cooldown Reduction**. Semakin cepat unit Anda dapat melakukan spam kemampuan, semakin kecil kemungkinan musuh lolos dari pertahanan Anda.

### Daftar Prioritas Peningkatan
1.  **Pengurangan Cooldown:** Esensial untuk menjaga aliran serangan yang konstan.
2.  **Jangkauan Kemampuan:** Meningkatkan "zona bunuh" untuk unit AoE Anda.
3.  **Pengganda Kerusakan:** Diperlukan untuk gelombang akhir di mana HP musuh meningkat secara signifikan.

## Mengoptimalkan Perangkat Keras Anda untuk Farming 24/7

Farming di Anime Apocalypse bisa membebani perangkat keras Anda. Jika Anda berencana membiarkan PC Anda menjalankan makro yang ditemukan di **discord Anime Apocalypse** semalaman, pertimbangkan optimasi berikut:

*   **Pengaturan Grafis:** Turunkan grafis Roblox Anda ke level 1. Ini mengurangi beban pada GPU Anda dan mencegah crash selama gelombang dengan banyak partikel.
*   **Prioritas Task Manager:** Atur Roblox ke "High Priority" di Task Manager Windows Anda untuk memastikan perangkat lunak makro (TinyTask) tidak mengalami input lag.
*   **Display Sleep:** Nonaktifkan mode tidur monitor Anda, tetapi jangan ragu untuk mematikan monitor secara manual. Makro akan terus berjalan selama OS tetap aktif.

Untuk dukungan teknis lebih lanjut dan build game terbaru, Anda dapat mengunjungi [Platform Resmi Roblox](https://www.roblox.com/) untuk memeriksa pembaruan game atau pengumuman pengembang.

## Pemilihan Peta Strategis

Meskipun makro yang dibagikan di **discord Anime Apocalypse** secara teknis dapat bekerja di berbagai peta, komunitas umumnya menyetujui satu peta "terbaik" untuk farming infinite. Anda menginginkan peta dengan jalur lurus atau putaran yang dapat diprediksi yang tidak mengganggu kemampuan makro untuk "berjalan ke depan" guna membersihkan lapangan awal.

*   **Peta yang Direkomendasikan:** Pilih peta dengan jalur garis lurus terpanjang. Ini meminimalkan kemungkinan musuh tersangkut di belakang objek lingkungan di mana unit Anda tidak dapat menjangkau mereka.
*   **Manajemen Lapangan:** Ingatlah bahwa ketika lapangan awal aktif, musuh mungkin tidak menyerang Anda. Makro Anda harus diprogram untuk memindahkan karakter Anda keluar dari zona ini segera setelah pertandingan dimulai.

## FAQ

**Q: Di mana saya bisa menemukan tautan ke discord Anime Apocalypse?**

A: Tautan tersebut biasanya ditemukan dalam deskripsi game di situs Roblox atau di bagian tautan sosial profil pengembang. Banyak kreator YouTube juga menautkan ke server komunitas pribadi mereka yang menghosting file makro tertentu.

**Q: Apakah menggunakan makro seperti TinyTask bisa dibanned?**

A: Sebagian besar pemain Anime Apocalypse menggunakan makro untuk farming Infinite Mode tanpa masalah. Namun, "scripting" (menggunakan eksploitasi untuk memodifikasi kode game) sangat dilarang dan akan mengakibatkan ban permanen. Selalu gunakan makro berbasis input daripada skrip injeksi kode.

**Q: Apa resolusi layar terbaik untuk makro Anime Apocalypse?**

A: Resolusi standar untuk hampir semua makro yang dibagikan komunitas adalah **1920 x 1080**. Jika Anda menggunakan monitor 1440p atau 4K, Anda mungkin perlu menyesuaikan pengaturan tampilan Anda ke 1080p sebelum memulai makro untuk memastikan klik terdaftar dengan benar.

**Q: Mengapa unit saya berhenti menyerang selama sesi AFK?**

A: Ini biasanya terjadi karena "lapangan" belum dibersihkan. Pastikan makro Anda menyertakan perintah untuk berjalan ke depan di awal setiap pertandingan. Selain itu, periksa apakah unit Anda diatur ke "Auto-Upgrade" atau makro tersebut mengklik tombol upgrade pada interval yang benar.
