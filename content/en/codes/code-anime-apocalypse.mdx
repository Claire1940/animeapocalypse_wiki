## Why Codes Fail (and How to Fix It Quickly)

Even valid codes can fail due to tiny technical issues. Don’t panic—troubleshoot in this order:

| Error Message / Problem | Most Likely Cause | Fix |
|---|---|---|
| **Invalid Code** | Typo, wrong capitalization, bad source | Re-copy from trusted source and retry |
| **Code Expired** | Time-limited reward has ended | Remove from list and move on |
| **Already Redeemed** | Previously claimed on account | No further action needed |

| **Reward not visible** | Claimed item type unclear | Check all tabs (currency, boosts, inventory) |

### Advanced Checks for Persistent Issues

If you still can’t claim a working entry:

These steps solve most player-side issues without needing support tickets.
