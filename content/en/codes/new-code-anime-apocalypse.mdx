A lot of players assume every failed redemption means the code is fake. In reality, most failures come from predictable issues you can fix in under a minute.

### Common redemption errors and fixes

| Error type | What it looks like | Quick fix |
|---|---|---|
| **Expired code** | “Invalid” or no reward | Try newest batch only; older lists expire quickly |
| **Typing mismatch** | One character off, wrong caps | Copy-paste directly from trusted source |
| **Extra spaces** | Code appears right but fails | Remove leading/trailing spaces |
| **Already redeemed** | No new reward granted | Track redeemed entries in notes |
| **Server delay** | Reward message is delayed | Rejoin server and check inventory/currency |
| **UI bug** | Button does nothing | Restart game client and retry |

If you’re repeatedly getting errors, test this order:

1. Rejoin a fresh server.
2. Paste code in plain text (no emojis or symbols around it).
3. Redeem one code at a time.
4. Confirm reward increase after each submit.
